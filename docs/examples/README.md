# Example Data & Diagrams

This folder contains real-world example records, payloads, error scenarios, and workflow diagrams for reference when developing or debugging.

- `airtable_record_example.json`: Example Airtable record with image metadata
- `workflow_image_metadata_example.json`: Example normalized image metadata
- `shopify_payload_example.json`: Example Shopify sync payload
- `ebay_payload_example.json`: Example eBay sync payload
- `error_scenarios.md`: Common error cases and resolutions
- `mermaid_workflow_diagram.md`: Workflow and error handling diagrams
- `mermaid_sync_conflict_resolution.md`: Sync conflict resolution flow
